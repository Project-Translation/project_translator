# Project Translator

VSCode extension for project localization with shared monorepo core.

## Install

- Marketplace: `techfetch-dev.project-translator`
- Open VSX: `https://open-vsx.org/extension/techfetch-dev/project-translator`

## Development

```bash
npm install
npm run build
npm run package
```

Repository: `https://github.com/Project-Translation/project_translator`
