## طلب ترجمة المشروع

إذا كنت ترغب في المساهمة في ترجمة أو تحتاج إلى ترجمة مشروع:

1. أنشئ قضية باستخدام القالب التالي:

```md
**Project**: [project_url]
**Target Language**: [target_lang]
**Description**: وصف موجز لسبب قيمة هذه الترجمة
```

2. تدفق العمل:

```mermaid
sequenceDiagram
  Contributor->>Project Translator: Create translation issue
  Project Translator->>Community: Review issue
  Community-->>Contributor: Approve/Comment
  Contributor->>New Project: Start translation
  Contributor->>New Project: Submit to New Project
  Contributor->>Project Translator: Create Pull Request, modify README.Samples
  Project Translator-->>Project Translator: Review & Merge
```

3. بعد دمج الطلب، سيتم إضافة الترجمة إلى قسم العينات.

الترجمات الحالية قيد التنفيذ: [View Issues](https://github.com/Project-Translation/project_translator/issues)

## الميزات

- 📁 دعم الترجمة على مستوى المجلد
  - ترجمة مجلدات المشروع بأكملها إلى عدة لغات
  - الحفاظ على هيكل المجلد الأصلي وتسلسله
  - دعم الترجمة التراكمية للمجلدات الفرعية
  - اكتشاف تلقائي للمحتوى القابل للترجمة
  - معالجة دفعية للترجمات على نطاق واسع بكفاءة
- 📄 دعم الترجمة على مستوى الملف
  - ترجمة ملفات فردية إلى عدة لغات
  - الحفاظ على هيكل الملف الأصلي وتنسيقه
  - دعم لأوضاع الترجمة سواء للمجلدات أو الملفات
- 💡 ترجمة ذكية باستخدام الذكاء الاصطناعي
  - الحفاظ التلقائي على سلامة هيكل الرمز
  - ترجمة تعليقات الرمز فقط، مع الحفاظ على المنطق البرمجي
  - الحفاظ على تنسيقات JSON/XML وغيرها من هياكل البيانات
  - جودة ترجمة احترافية للوثائق التقنية
- ⚙️ تكوين مرن
  - تكوين مجلد المصدر وعدة مجلدات هدف
  - دعم لفواصل الترجمة المخصصة للملفات
  - تحديد أنواع الملفات المحددة للإهمال
  - دعم لخيارات نماذج الذكاء الاصطناعي المتعددة
- 🚀 عمليات سهلة الاستخدام
  - عرض تقدم الترجمة في الوقت الفعلي
  - دعم لإيقاف مؤقت/استئناف/إيقاف الترجمة
  - صيانة تلقائية لهيكل المجلد الهدف
  - ترجمة تدريجية لتجنب العمل المتكرر

## التكوين

يدعم التمديد الخيارات التالية للتكوين:

```json
{
  "projectTranslator.specifiedFolders": [
    {
      "sourceFolder": {
        "path": "مسار مجلد المصدر",
        "lang": "رمز لغة المصدر"
      },
      "destFolders": [
        {
          "path": "مسار مجلد الهدف",
          "lang": "رمز لغة الهدف"
        }
      ]
    }
  ],
  "projectTranslator.specifiedFiles": [
    {
      "sourceFile": {
        "path": "مسار ملف المصدر",
        "lang": "رمز لغة المصدر"
      },
      "destFiles": [
        {
          "path": "مسار ملف الهدف",
          "lang": "رمز لغة الهدف"
        }
      ]
    }
  ],
  "projectTranslator.currentVendor": "openai",
  "projectTranslator.vendors": [
    {
      "name": "openai",
      "apiEndpoint": "عنوان URL لنقطة الوصول إلى الواجهة البرمجية",
      "apiKey": "مفتاح مصادقة الواجهة البرمجية",
      "model": "اسم النموذج المستخدم",
      "rpm": "الحد الأقصى للطلبات في الدقيقة",
      "maxTokensPerSegment": 4096,
      "timeout": 30,
      "temperature": 0.0
    }
  ]
}
```

تفاصيل التكوين الرئيسية:

| خيار التكوين                            | الوصف                                                                                    |
| --------------------------------------- | -------------------------------------------------------------------------------------- |
| `projectTranslator.specifiedFolders`    | مجلدات مصدر متعددة مع مجلدات الوجهة المقابلة للترجمة                           |
| `projectTranslator.specifiedFiles`      | ملفات مصدر متعددة مع ملفات الوجهة المقابلة للترجمة                               |
| `projectTranslator.translationIntervalDays`     | فاصل الترجمة بالأيام (الافتراضي 7 أيام)                                              |
| `projectTranslator.ignoreTranslationExtensions` | قائمة بامتدادات ملفات النص التي لا تحتاج إلى ترجمة، ستتم نسخ هذه الملفات مباشرة  |
| `projectTranslator.ignorePaths`                 | قائمة بأنماط المسارات المهملة باستخدام نحو glob، لن يتم نسخ هذه الملفات                   |
| `projectTranslator.currentVendor`               | المزود الحالي للواجهة البرمجية المستخدم                                                                      |
| `projectTranslator.vendors`                     | قائمة تكوين المزودين للواجهة البرمجية                                                                  |
| `projectTranslator.systemPrompts`               | مصفوفة الإرشادات النظامية لتوجيه عملية الترجمة                                        |
| `projectTranslator.userPrompts`                 | مصفوفة الإرشادات المحددة من قبل المستخدم، ستتم إضافتها بعد الإرشادات النظامية أثناء الترجمة |
| `projectTranslator.segmentationMarkers`         | علامات التقسيم المكونة حسب نوع الملف، تدعم التعبيرات المنظمة                     |

## الاستخدام

1. افتح قائمة الأوامر (Ctrl+Shift+P / Cmd+Shift+P)
2. اكتب "Translate Project" واختر الأمر
3. إذا لم يتم تكوين مجلد المصدر، سApopup لتحديد المجلد
4. انتظر حتى اكتمال الترجمة

أثناء الترجمة:

- يمكن إيقاف الترجمة مؤقتًا/استئنافها عبر أزرار شريط الحالة
- يمكن إيقاف عملية الترجمة في أي وقت
- يتم عرض تقدم الترجمة في منطقة الإخطارات
- يتم عرض السجلات التفصيلية في لوحة الإخراج

## ملاحظات

- تأكد من وجود رصيد كافٍ لاستخدام الواجهة البرمجية
- يُفضل اختبارها أولاً مع مشاريع صغيرة
- استخدم مفاتيح واجهة برمجية مخصصة وحذفها بعد الانتهاء
